If you're certain that you won't have to owe taxes, i.e. you're getting money back then you don't have to meet the April deadline. I don't think that's exactly right. Here's how I understand it: If you owe money, you can file for an extension, which is automatically granted. However, you have to estimate the amount of taxes you owe, and if you don't send a check for that amount, you can get charged penalties (which are a percentage of what you owe based on how long it takes you to pay the whole thing -- not huge, but they can add up). If you do not owe money, you still have to file for the extension, but there aren't going to be any penalties.
